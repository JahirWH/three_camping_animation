Mi Página de Práctica

Integrando Three.js con HTML

Bienvenido a mi sitio

Esta es una página simple para practicar la integración de un modelo 3D (en este caso, un tipi con fogata y troncos) usando Three.js. El objetivo es aprender cómo mezclar contenido web normal con gráficos 3D.

Aquí debajo estará el contenedor donde se renderizará la escena de camping. Este espacio es completamente transparente para que puedas ajustar tu escena como si fuera una imagen PNG animada.

Contenido adicional

Puedes agregar más texto, imágenes, botones, secciones o lo que quieras.

Lo importante es que tu canvas 3D se integra dentro del flujo normal del documento HTML.
